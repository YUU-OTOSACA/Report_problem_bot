Report Problem NC

Телеграм-бот для сбора данных о проблемах оборудования. 
Бот присылает обращения пользователей администратору бота, и сохраняет их в гугл-таблицу.


Токен бота, id администратора бота, имя гугл-таблицы, куда сохраняются обращения, путь к файлу json, находятся в файле: ...\Report_problem_bot\config\bot.ini


Для запуска бота необходимо запустить через терминал файл bot.py*
Для работы бота необходимо интернет-соединение


Для создания бота использовался язык программирования python и библиотеки: aiogramm, pandas, gspread, google-api-python-client, oauth2client.


Инструкция по использованию бота:

1. Запустите бота и не закрывайте открывшееся окно.*

2. Зайдите в Telegram в чат с ботом.

3. Пропишите команду "/start".

4. Используёте кнопку "Сообщить о проблеме".

5. Напишите свои ФИО.

6. Опишите вашу проблему. 

7. Проверьте ваше обращение, если всё в порядке нажмите кнопку "Всё в порядке", если что-то не так нажмите кнопку "Изменить" и повторите шаги 5-7.

8. Ваше обращение отправлено администратору бота и сохраненно в гугл-таблицах, при возникновении новых проблем повторите шаги 4-7.


* - для компьютера, который является сервером.

Внимание! Для внесение данных о проблемах в таблицу, нужно создать сервисный аккаунт и выдать ему возможность редактировать таблицу. Подробнее можно ознакомиться здесь: https://dvsemenov.ru/google-tablicy-i-python-podrobnoe-rukovodstvo-s-primerami/ (раздел "Настройка подключения в Google Api Console")
